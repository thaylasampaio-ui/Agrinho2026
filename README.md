 Sobre o projeto

Este projeto foi desenvolvido como um site educativo sobre o Agrinho 2026, com o objetivo de apresentar informações sobre o programa e sua importância para a sociedade

O site foi feito utilizando HTML, CSS e JavaScript, de forma simples e acessível, pensando tanto em computadores quanto em celulares

O que é o Agrinho 2026?

O Agrinho 2026 é um programa educativo desenvolvido pelo SENAR (Serviço Nacional de Aprendizagem Rural) em parceria com escolas.Ele promove o aprendizado sobre o meio ambiente e a relação entre o campo e a cidade, incentivando atitudes mais conscientes e responsáveis

Objetivos do projeto

Os principais objetivos do Agrinho são: 
Incentivar o respeito ao meio ambiente, ensinar a importância da agricultura para a sociedade,
estimular a criatividade e a pesquisa promover o trabalho em equipe, incentivar pequenas atitudes que ajudam a construir um mundo melhor

Funcionalidades do site

O site possui algumas funções interativas:
Modo claro e modo escuro, aumentar e diminuir o tamanho da fonte (acessibilidade) e Layout adaptado para celular e computador

Tecnologias utilizadas

HTML, CSS, JavaScript

Estrutura do projeto
index.html: estrutura principal do site
style.css: estilização e visual do site
script.js: funções de interação (modo escuro, fonte etc.)

Texto do projeto

Os textos foram escritos com base em pesquisas sobre o Agrinho e adaptados, com foco em explicar de forma simples a importância do programa
